NeoFinder

File browser, command runner & editor toolkit for Vim

Vim 8+ Neovim Windows Linux macOS Python 3 TOML Config

Install

git clone https://github.com/mefardales/neofinder.git ~/.vim/pack/plugins/start/neofinder

vim-plug: Plug 'mefardales/neofinder'

One-liner: curl -fsSL https://raw.githubusercontent.com/mefardales/neofinder/main/install.sh | bash

Features

File Browser

Navigate directories, fuzzy search, glob patterns (*.py), path navigation (~/). Background indexing with Python for instant search.

Command System

Every command is a .py + .toml pair. STDIN/STDOUT/STDERR standard I/O. Create commands from the editor with templates.

TOML Configuration

~/.neofinder/config.toml with comments. Editor settings, themes, keybindings, paths. Auto-reload on save.

Tag Groups

Bookmark files into named groups. Browse by group. Tag from the finder with Ctrl-T.

Directory Favorites

Bookmark directories for quick access. Auto-cd follows the active file.

Themes

Matrix, Dark, Cyberpunk, Default. Affects the entire editor: syntax, statusline, finder.

Window Management

Split, resize, navigate windows. Terminal keybindings. Shift+Tab cycles windows.

Custom Keybindings

Bind any key to Vim commands or NeoFinder commands from config.toml. <Leader>nf namespace.

>> _
[PALETTE] 8/8 | glob | multi:0 [Tab] editor
> Browse :ff file browser
  Favorites :fv bookmarked directories
  Buffers :fb open buffers
  Tags :fg tagged file groups
  Terminal :fR open terminal
  Run :fr execute commands
  Commands :fe edit/create commands
  Config :fc config.toml

Command System

Every command is two files: a .toml handler and a .py script.

net_scan.toml
# NetScan -- ping, DNS, port scan a host

name = "NetScan"
desc = "Ping sweep or port check on a host"
deps = ["input", "output", "shell"]
out = "[Scan ${host}]"

[in]
host = "Host/IP: "
net_scan.py
STDOUT.print("Scanning %s..." % host)
stdout, stderr, rc = nf.sh("ping -c 3 %s" % host)
STDOUT.write(stdout.splitlines())

Python API

STDIN / STDOUT / STDERRStandard I/O streams
nf.sh(cmd)Run shell command
nf.buf.lines / .write() / .append()Buffer read/write
nf.input() / .confirm() / .select()User interaction
nf.tags.groups() / .files() / .add()Tag system
nf.open() / .vsplit() / .split()File operations
nf.echo() / .warn() / .error()Messages

Configuration

All settings in ~/.neofinder/config.toml. Auto-reload on save.

# Theme
[theme]
name = "matrix"

# Finder
[finder]
height = 15
preview = true
max_files = 50000

# Editor
[editor]
line_numbers = false
tabstop = 4
autochdir = true

# Custom keybindings
[keybindings.commands]
a = "NetInfo"     # <Leader>nfa
b = "NetScan"     # <Leader>nfb
