NeoFinder

File browser, command runner and editor toolkit for Vim/Neovim.
Pure Vimscript core + Python commands. Works on Linux, macOS, Windows.

1. Installation

vim-plug

Plug 'mefardales/neofinder'

Vim 8+ native

git clone https://github.com/mefardales/neofinder.git \
    ~/.vim/pack/plugins/start/neofinder

Neovim

git clone https://github.com/mefardales/neofinder.git \
    ~/.local/share/nvim/site/pack/plugins/start/neofinder

One-liner

curl -fsSL https://raw.githubusercontent.com/mefardales/neofinder/main/install.sh | bash

4. File Browser

Open with <Leader>ff. Lists the current directory. Type to search.

  • Fuzzy search — type letters in order, matches anywhere
  • Glob patterns — *.py, *.toml, test_*
  • Path navigation — type ~/, /etc/, ../ to jump
  • Cached — directory listings are cached, navigation is instant
  • Background indexing — Python indexes the full tree on first search (lazy, non-blocking)
  • Auto-cd — when autochdir = true in config, cwd follows the active file

6. Command System

Every command is two files: a .toml handler (contract) and a .py file (logic).

Create a command

  1. Open palette → Commands :fe → [+] New command
  2. Enter name (e.g. DiskUsage)
  3. Both files open in vsplit. The .py has a full API reference in comments.

Or create files manually in ~/.neofinder/python/. Filename becomes command name: disk_usage.py → DiskUsage.

Handler (.toml)

# NetScan -- ping, DNS, port scan a host

name = "NetScan"
desc = "Port scan a host"
deps = ["input", "output", "shell"]
out = "[Scan ${host}]"

[in]
host = "Host/IP: "

Logic (.py)

STDOUT.print("Scanning %s..." % host)
stdout, stderr, rc = nf.sh("ping -c 3 %s" % host)
STDOUT.write(stdout.splitlines())

if rc != 0:
    STDERR.print("Host unreachable")

Handler fields

FieldTypeDescription
namestringCommand name (PascalCase)
descstringDescription shown in palette
depsarray"input", "output", "shell", "buffer", "tags"
outstringOutput buffer title. Supports ${var}
pipestring"buffer" to load current buffer into STDIN
[in]sectionVariables to ask the user. key = "prompt"

Standard I/O

Every command gets three streams:

StreamUsageDestination
STDIN.text, .lines, .varnameInput from handler [in] or pipe
STDOUT.print(), .write()Auto-flushed to output buffer
STDERR.print(), .write()Auto-shown as errors

7. Python API

Available as nf.* inside every .py command.

Context

nf.file              # current file full path
nf.filename          # file name only
nf.dir               # working directory
nf.line              # current line text
nf.filetype          # file type
nf.theme             # active theme name

Buffer

nf.buf.lines         # all lines as list
nf.buf.text          # full buffer as string
nf.buf.name          # buffer filename
nf.buf.line          # current line
nf.buf.line_number   # current line number
nf.buf.selection     # visual selection

nf.buf.write(x)      # replace buffer content
nf.buf.append(x)     # append to buffer
nf.buf.insert(x, n)  # insert at line n
nf.buf.clear()       # clear buffer
nf.buf[0]            # read line by index
nf.buf[0] = "x"      # write line by index

Shell

stdout, stderr, rc = nf.sh("cmd")
nf.sh_output("cmd")           # run & show in output buffer
nf.sh_lines("cmd")            # returns list of lines

User input

nf.input("prompt")             # ask for text
nf.confirm("sure?")            # yes/no dialog
nf.select(["a", "b"])          # pick from list

Tags

nf.tags.groups()               # list group names
nf.tags.files("group")         # files in group
nf.tags.add(path, "group")     # tag a file
nf.tags.remove(path)           # untag

Files

nf.open(path)                  # edit file
nf.vsplit(path)                # vertical split
nf.split(path)                 # horizontal split
nf.scratch(lines, "title")     # new scratch buffer
nf.buffers()                   # list open buffers

Messages

nf.echo("msg")                 # green message
nf.warn("msg")                 # yellow warning
nf.error("msg")                # red error

8. Configuration

All settings in ~/.neofinder/config.toml. Open with <Leader>fc. Saves with :w and reloads instantly.

# Theme: "matrix", "dark", "cyberpunk", "default"
[theme]
name = "matrix"

# Finder panel
[finder]
height = 15
preview = true
preview_width = 60
max_files = 50000

# Statusline
[statusline]
enabled = true

# Editor
[editor]
line_numbers = false
wrap = true
tabstop = 4
expandtab = true
encoding = "utf-8"
autochdir = true
hidden = true
swapfile = false
undofile = true
splitright = true
splitbelow = true
mouse = "a"
clipboard = "unnamedplus"

# Search
[search]
ignorecase = true
smartcase = true
hlsearch = true

# Terminal
[terminal]
shell = ""                     # empty = system default

# Auto-actions on save
[on_save]
trim_whitespace = true
final_newline = true

# Ignore patterns
ignore = [".git", "node_modules", "__pycache__"]

# Paths
[paths]
tags = "~/.neofinder/tags"
commands = "~/.neofinder/python"
undodir = "~/.neofinder/undo"

# Keybindings
[keybindings]
enabled = true

# Custom Vim key mappings
[keybindings.map]
# Ctrl+s = ":w<CR>"
# F5 = ":!python3 %<CR>"

# NeoFinder command bindings (single letter = <Leader>nf + letter)
[keybindings.commands]
# a = "NetInfo"                # <Leader>nfa
# b = "NetScan"                # <Leader>nfb

10. Project Structure

neofinder/
├── plugin/neofinder.vim              # Commands, mappings, startup
├── autoload/neofinder.vim            # Palette, browse, backend detection
├── autoload/neofinder/
│   ├── core.vim                      # Finder UI, input loop, fuzzy/glob filter
│   ├── theme.vim                     # Global themes
│   ├── preview.vim                   # File preview pane
│   ├── config.vim                    # TOML config loader + auto-reload
│   ├── statusline.vim                # Powerline statusline
│   ├── buffers.vim                   # Buffer manager, terminal
│   ├── python.vim                    # Command system (.py + .toml)
│   ├── tags.vim                      # Tags, favorites
│   ├── sources.vim                   # Data gathering for all sources
│   ├── actions.vim                   # Open, split, tag, execute actions
│   ├── indexer.vim                   # Vim wrapper for Python indexer
│   ├── indexer.py                    # Fast file indexer (os.walk)
│   ├── runtime.py                    # Python runtime (nf.*, STDIN/STDOUT/STDERR)
│   ├── toml_parser.py                # TOML parser for config + handlers
│   └── commands/                     # Built-in commands (.toml + .py pairs)
├── docs/index.html                   # This page
├── install.sh                        # One-click installer
└── README.md
