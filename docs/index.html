capabilities

Everything you need, inside Vim

Eight integrated tools. One coherent plugin. Zero context switches.

📁
File Browser
Navigate directories with fuzzy search and glob patterns. Background Python indexing for instant results across large trees.
*.py glob ~/ nav indexed
⚙
Command System
Every command is a .py + .toml pair. Standard STDIN/STDOUT/STDERR I/O streams. Create and edit commands directly from the editor.
.toml + .py stdio scriptable
📝
TOML Config
All settings in ~/.neofinder/config.toml with inline comments. Auto-reloads on save — no restart needed.
hot-reload commented
🏷
Tag Groups
Bookmark files into named groups. Browse by group. Tag from the finder with Ctrl-T, untag with Ctrl-D.
named groups Ctrl-T
⭐
Favorites
Bookmark directories for quick access. Auto-cd follows the currently active file as you navigate buffers.
auto-cd bookmarks
🌐
Themes
Matrix, Dark, Cyberpunk, Default. Theme affects the entire editor: syntax highlighting, statusline, and finder chrome.
Matrix Cyberpunk Dark
⚙
Window Management
Split, resize, and navigate windows with dedicated keybindings. Terminal integration. Shift+Tab cycles through open panes.
splits resize terminal
⌨
Custom Keybindings
Bind any key to Vim commands or NeoFinder commands from config.toml. Uses the <Leader>nf namespace.
<Leader>nf configurable

scripting

Command System

Every command is a pair of files: a .toml descriptor and a .py script. Write them in any editor, run them from inside Vim.

net_scan.toml TOML 
1# NetScan -- ping, DNS, port scan
2
3name = "NetScan"
4desc = "Port scan a host"
5deps = ["input", "output", "shell"]
6out  = "[Scan ${host}]"
7
8[in]
9host = "Host/IP: "
net_scan.py Python 
1STDOUT.print("Scanning %s..." % host)
2
3stdout, _, rc = nf.sh(
4    "ping -c 3 %s" % host
5)
6
7STDOUT.write(stdout.splitlines())

configuration

config.toml

One file. Human-readable. Auto-reloads on save.

~/.neofinder/config.toml

All NeoFinder settings live in a single TOML file with full comment support. Modify it, save it, and the changes take effect immediately — no :source, no restart.

  • [theme] — Switch between Matrix, Dark, Cyberpunk, or Default themes
  • [finder] — Panel height, preview toggle, max file index size
  • [editor] — Line numbers, tab width, autochdir, and other Vim settings
  • [keybindings.commands] — Map single letters to commands under <Leader>nf
  • Auto-reload — Changes apply instantly when you save the file
  • Comments — Full TOML comment support; annotate every setting
~/.neofinder/config.toml TOML 
 1[theme]
 2name = "matrix"
 3
 4[finder]
 5height     = 15
 6preview    = true
 7max_files  = 50000
 8
 9[editor]
10tabstop    = 4
11autochdir  = true
12
13# custom keybindings under <Leader>nf
14[keybindings.commands]
15a = "NetInfo"   # <Leader>nfa
16b = "NetScan"   # <Leader>nfb

getting started

Install NeoFinder

Works with Vim 8+ and Neovim. Requires Python 3 with pynvim for Neovim or vim-python3 for Vim.

📦 vim-plug
Plug 'mefardales/neofinder'
Add to your ~/.vimrc or init.vim, then run :PlugInstall
💾 Vim 8+ / Neovim packages
git clone https://github.com/mefardales/neofinder ~/.vim/pack/plugins/start/neofinder
Native Vim packages. No plugin manager required. Neovim path: ~/.local/share/nvim/site/pack/
🚀 One-liner (curl)
curl -fsSL https://raw.githubusercontent.com/mefardales/neofinder/main/install.sh | bash
Detects your setup and installs automatically. Inspect the script before running: view source
